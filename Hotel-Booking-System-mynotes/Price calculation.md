
# Price Calculation

## Overview
This document outlines the pricing methodology for hotel bookings.

## Calculation Formula

```
Total Price = (Nightly Rate × Number of Nights) + Taxes + Fees
```

## Components

### Base Rate
- Determined by room type and season
- Adjusted for demand and occupancy

### Taxes
- Local and state taxes applied to base rate
- Varies by location

### Additional Fees
- Resort fees
- Service charges
- Cleaning fees (if applicable)

## Examples

| Room Type | Nightly Rate | Nights | Subtotal | Taxes | Total |
|-----------|-------------|--------|----------|-------|-------|
| Standard  | $100        | 3      | $300     | $30   | $330  |
| Deluxe    | $150        | 3      | $450     | $45   | $495  |
